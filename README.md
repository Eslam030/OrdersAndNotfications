# OrdersAndNotifications

A Spring Boot REST API project designed to manage orders and notifications within a modular system architecture.  
The project focuses on backend development concepts including RESTful APIs, layered architecture, database interaction, and object-oriented system design.

---

## Overview

This module was developed using **Spring Boot** to provide APIs for handling:

- Order Management
- Notification Services
- Backend Business Logic
- Database Operations
- Modular System Communication

---

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- REST API
- Maven
- MySQL
- Hibernate

---

## Architecture

The project follows a layered backend architecture:

```bash
src/
├── controller/
├── service/
├── repository/
├── entity/
├── dto/
├── config/
└── resources/
```

## Getting Started

### Prerequisites

Before running the project, make sure you have installed:

- Java JDK
- Maven
- MySQL
- IntelliJ IDEA / Eclipse

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Eslam030/OrdersAndNotfications.git
```

Navigate to the project directory:

```bash
cd OrdersAndNotfications
```

Install dependencies and run the project:

```bash
mvn spring-boot:run
```
